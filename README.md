# An Online Electronics Learning Platform



## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
npm install
npm start (or) node server.js

cd admin-dash
npm install
npm start

cd client-folder
npm install
npm start
```


## Tech Stack
 - MongoDB
 - ExpressJS
 - ReactJS
 - Antd
